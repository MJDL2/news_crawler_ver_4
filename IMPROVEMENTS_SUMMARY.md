## 알려진 문제

1. **날짜별 수집 기능 문제**
   - 대화형 모드에서 날짜별 수집 설정이 작동하지 않음
   - 해결법: `--daily` 옵션 사용

2. **Python 3.13 호환성**
   - cookiejar 모듈 오류 자동 패치 적용됨
   - 권장: Python 3.11-3.12 사용

3. **대화형 모드 제한사항**
   - 날짜별 수집 기능 미지원
   - 복수 검색어 기능 미지원
